# IpInfo <img width="150" align="right" src="https://github.com/bennyscripts/ipinfo/blob/main/IpInfo/Assets.xcassets/AppIcon.appiconset/app-1.png?raw=true">
IpInfo is a native macOS app that allows you to geolocate/lookup an IP address straight from the menu bar.  
It uses MaxMind GeoLite2 databases to locally get location and organisation information.
